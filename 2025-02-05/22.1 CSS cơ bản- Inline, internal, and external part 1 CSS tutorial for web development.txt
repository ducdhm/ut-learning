CSS (Cascading Style Sheets): 
là một ngôn ngữ được sử dụng để tìm và định dạng lại các phần tử được tạo ra bởi ngôn ngữ đánh dấu (HTML)
 
nếu HTML đóng vai trò định dạng các phần tử trên website như việc tạo ra các đoạn văn bản, các tiêu đề, ...
thì CSS sẽ giúp chúng ta có thể thêm style vào các phần tử HTML đó như đồi bố cục, màu sắc trang, đồi màu chữ, font chữ,...

-Có 3 nơi có thể đặt CSS:
+ Inline (CSS nọi tuyến): <tag style="css />
+ Internal (CSS nội bộ): <style>css</style>
+ External (CSS bên ngoài): <link href="style.css"/>

1. Inline (CSS nội tuyến) là cách viết mã CSS ngay bên trong phần tử cơ bản HTML
 Nhược điểm: 
-Khó thay đổi, mỗi lần thay đổi phải thay đổi một cách riêng lẻ
-Không tái sử dụng được
-Khó quản lý
-Khó tách biệt cấu trúc
